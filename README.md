# spring_JsoupParserh
Парсер сайтов. Реализовано пока что только парсинг авито. 
В папке с проектом нужна папка outputPictures для сохранения картинок.
в файле resources/parser.properties поменять параметр avito.url на нужный.
Бесплатные прокси работают криво, поэтому использование не обязательно, но желательно.
